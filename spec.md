Сверстай лендинг, состоящий из следующих 7 блоков. Строго следуй структуре и текстам. Для навигации используй якорные ссылки.

SEO: Правильно используй заголовки H1, H2. Добавь meta-теги description и keywords (используй слова из предоставленного семантического ядра).

Стилизация:

Предоставь полностью рабочий код одного HTML-файла со встроенными CSS и JavaScript (для бургер-меню), который можно сразу развернуть на хостинге. Код должен быть чистым, хорошо прокомментированным.
